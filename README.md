## Requirements
### [cspec](https://github.com/mumuki/cspec)
Installation
1. `git clone https://github.com/mumuki/cspec.git`
1. `cd cspec`
1. `make`
1. `sudo make install`

## Encrypt this!
[![Foo](https://user-images.githubusercontent.com/38147472/105423146-c134b600-5c23-11eb-9b4b-d548007acf0d.png)](https://www.codewars.com/kata/5848565e273af816fb000449)
### Example
```python
encrypt_this("Hello") == "72olle"
encrypt_this("hello world") == "104olle 119drlo"
```
